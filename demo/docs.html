Get a list of users

Retrieves a paginated list of users.

Query Parameters

  • page

    integer

    Page number (optional)

  • limit

    integer

    Number of users per page (optional)

Header Parameters

  • Authorization (required)

    string

    An authorization token for authentication

Responses

200 Successful response

Response Schema: application/json

401 Unauthorized access

    string

    An authorization token for authentication

Response Schema: application/json
